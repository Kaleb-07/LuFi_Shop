A modern E-Commerce Web Application built with React, Vite, and Context API.
The platform allows users to browse products, add items to cart, manage orders, and complete purchases through a clean and responsive interface.🚀 Features

👤 User Features
Product browsing and search
Product detail page
Add to cart and remove from cart
Checkout system
User authentication (login & register)
Profile management
Order tracking
Multi-language support
Dark & Light theme toggle

🛍 Shopping Features
Product categories
Product ratings and reviews
Shopping cart management
Secure checkout
Toast notifications
